# V4.0 Tracking

Adds target track storage and APIs.

Added:

- `target_tracks`
- `target_track_points`
- `GET /tracks`
- `GET /tracks/{track_id}`
- `GET /tracks/{track_id}/points`
- `POST /tracks/{track_id}/close`
- `POST /localization-results/{result_id}/track`

Successful localization results can now create or update active target tracks.
