# V3.4 Hybrid Localization

Adds GCC-PHAT helpers for future waveform refinement.

Added:

- 16-bit PCM WAV loader
- GCC-PHAT lag estimator
- hybrid localization path guarded by `GCC_PHAT_ENABLED`

Default:

```text
GCC_PHAT_ENABLED=false
```

When disabled or unavailable, the backend continues to use timestamp TDOA or
fallback localization.
